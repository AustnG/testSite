IMAGES

Basic of using images in your page.

Image Tags
  • img
  • src - where the img can be found (much like href)
  • alt - text to display/read in place of image
  • height/width - display sizing control (beware of overuse of this)
*NOTE: No matter the display size of your image, whatever the ACTUAL size of your image is what will be downloaded. Make sure to be aware of this to maximize the performance of your page (lots of large file sizes may slow down your page's performance; especially on mobile).

For alternative image sizes
  • srcset - provides different sources (mobile, desktop, etc.) different rendering options
  • sizes - tells browser make it relative to the veiwport size (uses "vw descriptor as in "Veiwport"; ex. 25vw, 50vw)

Picture Tag
Used to select images based on media queries or things like supported image formats (jpg, png, etc.)
  • Can include "TYPE" - this tells the browser "if you support this image type, go ahead and display this image. If NOT here's another image you can use"
  • Also works for media queries (or detecting page sizes/viewport widths)
