METADATA

Expaning Link Metadata
Used to provide helpful data to users that are using a browser or other tools to interperate your page
  • Language
  • Relationship - how the doc is related to your page (ex. Table of Contents)
  • Content - (image, video, etc.)
  • Access Key - ref the link using keyboard shortcuts
  • Target (open in new window, tab, blank container, etc.)
This will open a link in a new window -or- tab
*This is based on user prefences in the browser settings

TABLES

***TABLES ARE NOT FOR FORMATTING YOU PAGES! They are better used for displaying data***
Defining Tables
  • CAPTION - tells user what this table is (good for screen readers)
  • HEADER - typically the top row in the table.
  • BODY - where all the data is displayed.
  • FOOTER - define things like agg. values, sums, avgs, etc...
Table Tags
  • tr - table row
  • th - table header (column)
  • td - table data
  • colspan/rowspan - used for a col/row that will span multiple rows (good for headers)
