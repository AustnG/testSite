Anchors & Links

Links, baby.

Creating anchors, linking to and within docs, and expanding metadata.

Source Anchor
[Uses href=""]
Takes you to the SOURCE of an anchor (href = "hypertext reference")
Named Anchor
[Uses name=""]
Provides a TARGET for the a tag to go to. Doesn't render a link//anything visable, but does create a target for you to be able to point to. Can link to this very specific point itself.
Implicit Anchor
[Uses id=""]
Can be used as a target as the item you want to nav to.
