Try to remember to use the 'for' attribute in the label tag to link a label to an input:

<form>
  <label for="recipie-name">Recipie Name</label>
  <input typel="text" id="recipie-name">
</form>

*Input fields are great for little bits of text.

For areas where you need a lot of text//input, use the TEXTAREA tag:

<form>
  <label for="ingredients">Ingredients</label>
  <textarea id="ingredients"><textarea>
</form>

Attribute Selectors are a way to style a tag based on the attributes:

input{
  width: 500px;
  margin-bottom: 500px;
}

input[type=submit] {
  width: 125px;
  font-size: 30px;
}
