What is Java?
It's a cross-platform programming language using the .java/.class file. Object oriented programming

-Perc of writing in Java is you can write the code one time, then run it ANYWHERE!
(on a "Java Virtual Machine" (JVM) which most all modern computers have)

Java Virtual Machine (JVM)
The software that runs the Java programs; compiled into '.class' files (avaliable on all major operating systems)
Integrated Development Environment (IDE)
Program to type in the source code (ex. Eclipes, Visual Studio, etc.)
Method
A block of code designed to perform a task; essentially the same thing as a 'Funtion' (or subroutine)
Class
A blueprint from which other objects are created (ex. 'Car' Class = type, size, weight, wheel type, color, top speed, etc.)
